	Joyería Morgan - Sistema de Gestión Académico

Este es un sistema integral de gestión para una joyería de lujo, desarrollado como proyecto académico utilizando Spring Boot y Thymeleaf.

	TECNOLOGIAS UTILIZADAS
1 Backend: Java 17 & Spring Boot.
2 Frontend: Thymeleaf, Bootstrap 5 & Font Awesome 6.
3 Base de Datos: MySQL.
4 Análisis: Chart.js para el Dashboard de ventas.

	MANEJO DE VERSIONES 
El proyecto sigue un control de versiones estricto mediante Git Tags:
1  Versión inicial estable con operaciones CRUD.
2  Mejora de Interfaz de Usuario (UI/UX), integración de iconos y Dashboard dinámico.
3. Optimización del Modelo: Actualización profunda de `Producto.java` y optimización de los formularios de registro para una mejor validación de datos.
4. Integración de Cambios (PR #1): Fusión de la rama `feature-ana-cambios` con las primeras mejoras estructurales.
5. Refactorización de Diseño: Se realizaron múltiples ajustes estéticos ("Modifique el diseño") para que la interfaz sea más limpia y profesional.
6. Consolidación de Estilos: Segunda fase de ajustes de diseño para asegurar la coherencia visual en todos los módulos (Ventas, Inventario, Clientes).
7. Fusión Final (PR #2): Integración total de las últimas mejoras de diseño y funcionalidad desde la rama de características a la rama principal (`master`).

	INSTALACION Y CONFIGURACION
1. Clonar el repositorio: `git clone https://github.com/peteralainsantosbruno-beep/joyeriaMorgan-academico.git`
2. Crear la base de datos `joyeriamorgan` en MySQL.
3. Ejecutar la aplicación desde el IDE (Eclipse/STS).
