# Joyería Morgan - Sistema de Gestión Académico

Este es un sistema integral de gestión para una joyería de lujo, desarrollado como proyecto académico utilizando **Spring Boot** y **Thymeleaf**.

## Tecnologías Utilizadas
* **Backend:** Java 17 & Spring Boot.
* **Frontend:** Thymeleaf, Bootstrap 5 & Font Awesome 6.
* **Base de Datos:** MySQL.
* **Análisis:** Chart.js para el Dashboard de ventas.

## Manejo de Versiones (SemVer)
El proyecto sigue un control de versiones estricto mediante Git Tags:
* **v1.1.0**: Versión inicial estable con operaciones CRUD.
* **v1.2.0**: Mejora de Interfaz de Usuario (UI/UX), integración de iconos y Dashboard dinámico.

## Instalación y Configuración
1. Clonar el repositorio: `git clone https://github.com/peteralainsantosbruno-beep/joyeriaMorgan-academico.git`
2. Importar el archivo `joyeriaMorgan.sql` en MySQL.
3. Ejecutar la aplicación desde el IDE (Eclipse/STS) o vía Maven: `mvn spring-boot:run`
